# Sentra

Sentra is a Fake Review and Fraudulent User Detection Framework.

## Features

- **Fraudulent User Detection:** Identifies anomalous user behaviors and flags suspicious accounts.
- **Fake Review Scoring:** Analyzes review patterns to detect automated or paid fake reviews.
- **Seller Trust Scoring:** Continuously evaluates seller integrity based on transactional and behavioral data.
- **Admin Dashboard:** A moderation queue and reporting interface for platform administrators.

## Project Structure

- `backend/`: Python-based API and scoring engine.
- `frontend/`: Next.js-based web application for customers, sellers, and admins.
- `database/`: SQL schemas and seed data.
- `documentation/`: System architecture and API documentation.
